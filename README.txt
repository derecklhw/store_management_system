Installations Package
Below packages to be installed though pip install or directly through pycharm.
- matplotlib version 3.5.1
- mysql-connector version 2.2.9
- mysql-connector-python version 8.0.28

How to run
- First amend host, user and password in the database.py and the server.py as per your mysql workbench connection.
- Run database.py to create the database schema.
- Run database.py again to create the tables and insert their respective records.
- Run server.py
- Run client.py
